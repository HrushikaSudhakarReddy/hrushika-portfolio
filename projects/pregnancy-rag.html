Project

Pregnancy Coach RAG

LangChain, ChromaDB, FastAPI, Retrieval evaluation, Safety gating
Overview

An LLM-powered retrieval-augmented generation system designed for grounded answers with citations, built as a production-style chat API.

Business problem

LLM responses can be inconsistent and can hallucinate without grounding. The goal was to build a system that retrieves the right evidence, generates reliable answers with citations, and measures quality using evaluation metrics rather than subjective spot-checking.

What I built
  • Built a RAG pipeline with citations, memory, and reranking, exposed through a production-style /chat API (LangChain, ChromaDB, FastAPI).
  • Designed an intent and safety classifier and evaluated on 200+ queries using a confusion matrix.
  • Measured retrieval effectiveness with retrieval metrics (Recall@K, MRR) and iterated using repeatable evaluation runs.
  • Tuned chunking strategy, top-k retrieval, and reranking to improve relevance and reduce hallucination.
Results
  • Improved answer grounding by enforcing citation-based responses from retrieved sources.
  • Made reliability measurable via classifier evaluation and retrieval metrics, enabling structured iteration.
  • Increased relevance through systematic chunking and reranking experiments.
